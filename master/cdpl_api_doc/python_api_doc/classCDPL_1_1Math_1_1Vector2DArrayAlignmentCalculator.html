Returns the numeric identifier (ID) of the wrapped C++ class instance.

Different Python Vector2DArrayAlignmentCalculator instances may reference the same underlying C++ class instance. The commonly used Python expression a is not b thus cannot tell reliably whether the two Vector2DArrayAlignmentCalculator instances a and b reference different C++ objects.
