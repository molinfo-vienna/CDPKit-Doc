Initializes the table with default colors for frequently occurring chemical elements.

Currently, colors for the following elements are provided: C, H, N, O, S, P, F, Cl, Br, I, Li, Na, B, Fe, Ba, Mg, Zn, Cu, Ni, Ca, Mn, Al, Ti, Cr, Ag, Si, Au and He.
