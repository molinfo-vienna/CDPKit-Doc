◆ getObjectID()

int CDPL.Math.Matrix2UL.getObjectID ( )

Returns the numeric identifier (ID) of the wrapped C++ class instance.

Different Python Matrix2UL instances may reference the same underlying C++ class instance. The commonly used Python expression a is not b thus cannot tell reliably whether the two Matrix2UL instances a and b reference different C++ objects. The numeric identifier returned by this method allows to correctly implement such an identity test via the simple expression a.getObjectID() != b.getObjectID().

Returns
The numeric ID of the internally referenced C++ class instance.
