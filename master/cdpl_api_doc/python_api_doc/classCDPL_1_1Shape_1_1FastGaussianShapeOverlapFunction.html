Detailed Description

Shape.GaussianShapeOverlapFunction implementation that uses two approximation techniques to trade some accuracy for a substantial speedup compared to Shape.ExactGaussianShapeOverlapFunction.

Two independent acceleration features can be enabled:

  • Proximity optimization prunes Gaussian-product pair contributions based on a scaled van-der-Waals proximity test (see setRadiusScalingFactor()).
  • Fast exponential function replaces the expensive std::exp call with a fast approximation that is accurate enough for screening-style overlap evaluation.
