Detailed Description

Assigns numeric labels to specific atoms of a molecular graph that are described by substructure patterns.

Patterns are added via addPattern() (each pattern carries a query molecular graph, an atom-label value, a priority and match-handling flags). On execute() the typer iterates the registered patterns in priority order, runs each as a substructure query and assigns the corresponding atom label to every matched atom unless the atom has already received a label from a pattern with higher priority.

int CDPL.Chem.PatternAtomTyper.getObjectID ( )

Returns the numeric identifier (ID) of the wrapped C++ class instance.

Different Python PatternAtomTyper instances may reference the same underlying C++ class instance. The commonly used Python expression a is not b thus cannot tell reliably whether the two PatternAtomTyper instances a and b reference different C++ objects. The numeric identifier returned by this method allows to correctly implement such an identity test via the simple expression a.getObjectID() != b.getObjectID().

The numeric ID of the internally referenced C++ class instance.
