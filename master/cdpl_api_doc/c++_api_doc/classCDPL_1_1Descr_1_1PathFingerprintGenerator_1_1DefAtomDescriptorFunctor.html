CDPL::Descr::PathFingerprintGenerator::DefAtomDescriptorFunctor::DefAtomDescriptorFunctor ( unsigned int  flags = DEF_ATOM_PROPERTY_FLAGS)
inline

Constructs the atom descriptor functor object for the specified set of atomic properties.

The flags argument is an OR combination of the constants defined in namespace Chem::AtomPropertyFlag. Supported property flags are:

  • Chem::AtomPropertyFlag::TYPE
  • Chem::AtomPropertyFlag::ISOTOPE
  • Chem::AtomPropertyFlag::AROMATICITY
  • and Chem::AtomPropertyFlag::FORMAL_CHARGE
Parameters
flagsSpecifies the set of considered atomic properties.
