Detailed Description

Creates 2D structure diagrams of molecular graphs in the Portable Network Graphics (PNG) [WPNG] format.

PNGMolecularGraphWriter uses Vis::StructureView2D for the creation of 2D structure diagrams. All control-parameters and properties provided for the customization of Vis::StructureView2D are also supported by PNGMolecularGraphWriter.
