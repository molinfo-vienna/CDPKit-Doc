Detailed Description

template<std::size_t Dim, typename T, typename Derived>
class CDPL::Util::DGCoordinatesGeneratorBase< Dim, T, Derived >

Serves as foundation for subclasses that perform coordinates generation based on distance-geometry.

Template Parameters
DimThe dimension of the coordinates to generate.
TThe scalar value type to use for coordinates and constraint bounds.
DerivedThe type of the subclass.
