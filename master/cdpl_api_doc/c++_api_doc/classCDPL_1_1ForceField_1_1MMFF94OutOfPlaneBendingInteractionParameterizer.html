Detailed Description

Generates the MMFF94 out-of-plane bending interactions for the trigonal centers of a molecular graph.

For every atom j with exactly three neighbors that is a trigonal center (per the MMFF94 atom-type properties) the parameterizer generates three out-of-plane records — one for each permutation of the three neighbors as the out-of-plane atom l with the remaining two as in-plane terminal atoms. The required out-of-plane force constants are taken from the parameter table, falling back through the primary-to-parameter atom-type map when no exact entry is available.
