Detailed Description

Calculation of a 3D auto-correlation descriptor of a molecular graph partitioned by specific atom types.

Atom types considered for partitioning are:

  • Chem::AtomType::H
  • Chem::AtomType::C
  • Chem::AtomType::N
  • Chem::AtomType::O
  • Chem::AtomType::S
  • Chem::AtomType::P
  • Chem::AtomType::F
  • Chem::AtomType::Cl
  • Chem::AtomType::Br
  • Chem::AtomType::I
