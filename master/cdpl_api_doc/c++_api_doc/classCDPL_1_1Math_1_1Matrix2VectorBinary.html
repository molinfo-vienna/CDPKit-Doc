Detailed Description

template<typename E1, typename E2, typename F>
class CDPL::Math::Matrix2VectorBinary< E1, E2, F >

Expression-template node interpreting a binary combination of a vector expression E1 and a matrix expression E2 as a vector (e.g. vector-matrix product), via the per-element functor F invoked with both expressions and the index.

The resulting vector has size e2.getSize2() (i.e. the matrix expression's number of columns).

E1The vector expression type.
E2The matrix expression type.
FThe per-element functor type.
